# 📱✨小红书封面生成器

这是一款专为小红书内容创作者设计的自动化工具,旨在简化图文排版过程,提升创作效率。主要功能包括:
🖋 智能文字排版与分页
🎨 文字颜色与背景色自定义
📝 苹果备忘录风格+网格背景
✍️ 中英文手写字体支持
😊 Emoji表情集成(开发中)

我们致力于为创作者提供一个直观、高效的创作平台,让您专注于内容本身,而无需担心繁琐的排版细节。无论您是经验丰富的博主还是刚起步的新手,我们的工具都能帮您轻松制作出精美的小红书图文。
